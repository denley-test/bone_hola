Support Chinese hola

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'bone_hola'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install bone_hola

## Usage

    $ bone_hola
    $ bone_hola english
    $ bone_hola spanish
